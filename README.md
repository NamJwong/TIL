# TIL

## 작성 방법

- 데일리: `TIL/[YYMMDD].md`에 배운 점 작성
- 일요일마다: 한 주 동안 `TIL/`에 작성한 내용을 각종 폴더로 적절히 배치
  - 모든 내용 X. 기억해두면 좋을 내용들

## 어떤 기록을 해야 하는지

- 학습 내용을 체화시키는 데에 도움되어야 함
- 기억하기 좋은 형태여야 함
- 다시 돌아보았을 때 잘 읽혀야 함

## 위와 같은 기록이 되려면 어떻게 해야 하는지

- 나의 말로 작성
- 사례, 경험 적극 활용하여 작성
- 잘 압축하기. 두괄식 서술
  - 작성 중 '한마디로 말하자면?' 질문 많이 던지기
- 학습 내용을 말로 많이 꺼내보기
